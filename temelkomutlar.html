Turkish Roblox Devs - Temel Komutlar #2

Benim burda kastetmek istediğim print, aritmetik operatörler, if, else, elseif. bugün sizlere bunları burda yazı olarak nasıl yeteri kadar anlatılıyorsa anlatacağım.


Print


Print basit haliyle ekrana / konsola istediğimiz bir değeri yada yazıyı yazdırmaktır örnekleri aşağıda


print("Merhaba Dünya") -- konsol : Merhaba Dünya
local x = 10 -- x'i 10 olarak atadık
print(x) -- konsol : 10
local y = 5 -- y'yi 5 olarak atadık
print(y) -- konsol : 5
local z = false -- z'yi false olarak atadık
print(z) -- konsol : false
local a = true -- a'yı true olarak atadık
print(a) -- konsol : true


Arimetik Oparetörler


Arimetik opertatörleri hepimiz biliyoruz değilmi toplama,çıkarma,bolme ve çarpma daha fazlası farklı şekillerde var fakat şuan bunu işlemeyeceğiz. Aşağıda kodda nasıl aritmetik operatörleri kullanacağınızı gösterdim lütfen bi inceleyin.


local x = 5 -- x'i 5 olarak atadık
local y = 3 -- y'yi 3 olarak atadık
print(y + x) -- Toplama, konsola 5+3 yani 8 i yazdıracak
local x = 10 -- x'i 10 olarak atadık
local y = 2 -- y'yi 2 olarak atadık
-- Not : Biz kodda çarpma işaretini x veya . yerine * olarak kullanırız.
print(x*y) -- Çarpma, konsola 10 x 2 yani 20'yi yazdıracak
-- Not : Biz bölme işaretini kodda "/" olarak kullanırız
print(x/y) -- Bölme, konsola 10 bölü 2 yani 5 i yazdıracak
print(x-y) -- Çıkartma, konsola 10 - 2 yani 8 i yazdıracak


İf,else,elseif


İf türkçede eğer demektir, else ise "aksi halde" yada "yoksa" anlamlarına gelir, elseif ise sorgu cümlesidir birden fazla if yazmak yerine if ve elseif i kullanarak bir katmandan hepsini yönetebiliriz aşağıda örneklerini verdim.


local y = false
local x = true
local z = math.random(0,4)

if y == false then
print("y'nin değeri false")
else
print("y'nin değeri true")
end

if x == true then
print("x'in değeri true")
else
print("x'in değeri false")
end

if z == 1 then
print("z 1 çıktı!")
elseif z == 2 then
print("z 2 çıktı!")
elseif z == 3 then
print("z 3 çıktı!")
else
print("z 1 ile 3 arasında bir değer çıkmadı")
end
